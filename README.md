# Study Planner & Task Scheduler (DAA Project)

## Overview

This project is a Study Planner and Task Scheduler that helps organize courses based on time requirements, priority (credits), and prerequisite relationships. 
It applies core Design and Analysis of Algorithms (DAA) concepts to optimize study time and maximize productivity.

## Algorithms Used

* Greedy Algorithm for task scheduling
* Dynamic Programming (0/1 Knapsack) for optimal course selection
* Backtracking for exploring possible schedules
* Topological Sorting for handling prerequisites

## Features

* Efficient task prioritization based on credits
* Optimal utilization of available study time
* Handles course dependencies using graph-based approach
* Generates structured and optimized study schedules
* Menu-driven console application

## Tech Stack

* Language: C++
* Concepts: Design and Analysis of Algorithms
* Tools: g++, Visual Studio Code

## How to Run

1. Compile the program:
   g++ main.cpp

2. Run the executable:
   ./a.out

## Sample Input

* Course name
* Credits
* Hours required
* Prerequisites (if any)

## Output

* Scheduled course order
* Selected courses based on constraints
* Maximum achievable credits
